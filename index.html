CSS Visual Rules

Selector Name Description
font-family font family Defines the typeface of an element.
font-size font size Controls the size of text displayed.
font-weight font weight Defines how thin or thick text is displayed.
text-align text align Places text in the left, right, or center of its parent container.
color color Defines the color of the text.
background-color background color Defines the color behind the text.
opacity opacity Defines the transparency of an element/text.
background-image background image Allow to set the background of an element to an image with
!important important This flag will override any style, should almost never by used.
